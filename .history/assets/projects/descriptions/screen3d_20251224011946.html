~ Overview ~

This project turns a standard 2D monitor into a holographic window. By tracking the user's head position in real-time using stereo computer vision, the application dynamically distorts the 3D projection matrix. This creates an Off-Axis Projection, giving the illusion that the screen is a transparent pane of glass looking into a virtual world. Additionally, it can create holograms that appear to emerge from the screen when viewed with one eye closed or through a camera.

~ Features ~

The system features a custom Stereo Head Tracker using MediaPipe and OpenCV, a Calibration Suite for precise camera extrinsics, and a Transparent Overlay mode that integrates 3D objects seamlessly into the Windows desktop environment.
