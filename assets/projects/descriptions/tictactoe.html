TicTacToe

( MiniMax with α, β pruning )

#3 — 18/08/2025

~ Overview ~

This project implements a Tic Tac Toe AI using the Minimax algorithm with Alpha-Beta pruning. It allows you to play against an AI opponent, test strategies, and observe how decision trees work in practice. The engine evaluates all possible game states and prunes unneeded branches to achieve efficient, real-time decision making.

~ Features ~

You can choose whether the AI controls X or O, enable or disable move suggestions, and play interactively by entering coordinates or cell indices. The AI calculates the optimal move at each step and highlights it if suggestions are enabled. The algorithm balances efficiency and accuracy using Alpha-Beta pruning, making the game engine both fast and competitive.
